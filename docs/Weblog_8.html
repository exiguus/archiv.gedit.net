The Web Standarts Project

Das bereits seit 1998 existierende Web Standarts Projekt kämpft für einheitliche Web Standarts, die jedem Einzelnen einen schnellen und einfachen Zugang zu Web basierten Technologien erleichtern soll.

Webstandarts.org arbeitet hierbei mit Software Herstellern und Web Entwicklern zusammen, um eine stärkere Verbreitung von Web Standarts zu fördern. Welche die Kosten und Komplexität bei der Entwicklungen von Web Anwendungen minimieren und die Realisierbarkeit und Zugänglichkeit erhöhen.

Die Seite selbst, hält für den interessierten Besucher ein Menge an Artikeln, Tutorials, Howtos und Antworten rund um dieses Thema bereit.

<http://www.webstandards.org/>
