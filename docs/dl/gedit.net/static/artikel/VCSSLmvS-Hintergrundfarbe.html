Weniger Text

Das Problem(Javascript-Link) bei der Technik mit der Hintergrundfarbe, ist das man voraussagen muss welche Spalte die höhere sein wird.
In diesem Fall ist die rechte, blaue Box die voraussichtlich höhere, weshalb die linke Box einen transparenten Hintergrund hat und den darunterliegenden roten Wrapper zeigt.
Füllt man die Linke Spalte mit mehr Text(Javascript-Link), zerstört es das Layout.

Variable Schriftgröße(Javascript-Link) hingegen ist kein Problem.
